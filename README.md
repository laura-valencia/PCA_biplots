This script contains two function used to produce PCA biplots from the output of a DESeq2 (R) run on RNA-seq count data.
The functions include:
1) Obtain of a dds object to extract counts in a format that fits for plotting
2) Parsing a DESeq2 output file with differentially expressed genes, next to the previously obtained counts, to obtain a table with the top genes contributing to variation.
The function 2 also produces the PCA biplot.

For the writing of this software, Chat-GPT-UiO (gpt.uio.no) GPT-5.1 Thinking model was used to assist.
